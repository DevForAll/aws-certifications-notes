
# REPASAR
Highlights

- Infraestructura Global y Servicios de Cómputo: Dominas los conceptos de Regiones, Availability Zones y Edge Locations, así como el uso de servicios serverless como AWS Lambda y almacenamiento de larga duración en S3 Glacier.
- Seguridad y Accesos: Tienes un excelente entendimiento sobre la autenticación multifactor (MFA), el servicio AWS WAF para protección web y el cifrado mediante KMS.
- Beneficios de la Nube: Comprendes claramente conceptos fundamentales como Elasticidad, Agilidad y las Economías de Escala que ofrece AWS.

Focus areas

- Shared Responsibility Model (Modelo de Responsabilidad Compartida): Tuviste dificultades al diferenciar las responsabilidades en servicios de infraestructura (IaaS) como EC2. Recuerda que en EC2, el cliente es responsable del parcheo del sistema operativo, mientras que AWS se encarga del hardware y la red.
- Gobernanza y Monitoreo (CloudTrail vs. Config vs. Artifact): Es importante distinguir entre CloudTrail (registro de actividad/API), AWS Config (inventario y cambios de configuración) y AWS Artifact (informes de cumplimiento). Confundiste el registro de cambios con el de actividad.
- Gestión de Costos y Soporte: Necesitas revisar la diferencia entre AWS Budgets (alertas de presupuesto) y Cost Explorer (análisis de tendencias). También recuerda que solo el plan de soporte Enterprise incluye un TAM (Technical Account Manager).
- AWS Well-Architected Framework: Confundiste Excelencia Operativa con Fiabilidad (Reliability). La Fiabilidad es el pilar que se enfoca específicamente en la recuperación ante fallos y la disponibilidad.
